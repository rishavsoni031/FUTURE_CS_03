# API Security Risk Analysis

## 📌 Objective
To analyze the security of a public API and identify vulnerabilities.

## 🛠 Tools Used
- Postman
- Browser Developer Tools

## 🌐 API Tested
JSONPlaceholder API  
https://jsonplaceholder.typicode.com

## 🔍 Endpoints Tested
- /users
- /users/1
- /comments

## 🚨 Findings
- No Authentication (High)
- Data Exposure (Medium)
- Broken Access Control (High)
- Weak Rate Limiting (Low)
- Missing Security Headers (Low)
